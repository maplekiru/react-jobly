# Jobly

A mock job-board with listing of companies, jobs related to companies, all jobs and ability to search.

### Installing

* Download all dependencies
```
npm install
```

### Executing program

* Start application at localhost:3000
```
npm start
```

* Build Application for deployment
```
npm build
```
